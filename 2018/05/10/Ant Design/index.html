构建部署npm run build (入口文件构建到dist目录中)

IE8 （antd2不再支持）Reac有问题可访问[https://github.com/xcatliu/react-ie8](https://github.com/xcatliu/react-ie8)

四、Components

4.1 Button

四种类型：主按钮、次按钮、虚线按钮、危险按钮（主按钮同一区域最多一次，danger antd2.7支持）primary、default、dashed、danger
  • 尺寸：size：large大、small小、不设置-中
  • 加载中：loading属性
  • 按钮组合：放入Button.Group中
  • 图标：icon属性（也有Icon组件）
  • 不可用状态：disabled
  • 幽灵按钮：glost，背景透明，按钮内容反色antd2.7
属性 说明
glost/icon/loading/size/type/onClick 如上
href 行为与a一致，渲染为a标签
htmlType button原生type值
shape 按钮形状，circle圆形
target a链接属性target，href存在生效

除以上属性，额外直接传到button；渲染内容时，span会包围。

4.2 Icon

命名规范：实心/描线用-o区分；命名顺序[图标名]-[形状?]-[描线?]-[方向?]

使用：type,公网访问，可以[本地部署](https://ant.design/components/icon-cn/)
属性 说明
Spin 是否有旋转动画，false
Style 样式
Type 图标类型

4.3 Grid 栅格

24栅格系统，划分后的信息区称之为‘盒子’

行（row）、列（col）：col可以作为row的直接元素；.col-8 表示八格；超过24另起一行。

  • Flex 布局：
  • 对齐方式：align：top、middle、buttom、
  • Flex排序：order改变顺序
  • 响应式布局：xs/sm/md/lg/xl/xxl
  • 其他属性响应式：内嵌到 xs、sm等中 xs=
  • 栅格配置：间距 Row上gutter属性
Row属性 说明
align 垂直对齐方式top、middle、botton
gutter 间隔，可写成{ xs: 8, sm: 16 }
jstify flex布局水平排列方式：start/end/center/space-around/space-between
type 布局模式，flex
Col属性 说明
offset 向右偏移数
order 栅格顺序，flex布局有效
pull 向左移动的格数
push 向右

xs（576）sm（>576）md（>768）lg（>992）xl（>1200）xxl（>1600）

4.4 布局

  • Layout：布局容器，可嵌套如下和Layout本身

  • Header：顶部布局，可嵌套任何，只能放在Layout中

  • Sider：侧边栏，可嵌套任何，只能放在Layout中

  • Content：内容，可嵌套任何，只能放在Layout中

  • Footer：底部布局，可嵌套任何，只能放在Layout中

    | Layout属性 | 说明 |
    | ————— | ——————————- |
    | className/style | 通用 |
    | hasSider | 子元素有Sider，一般不指定，可用于服务端渲染时避免样式闪动 |

Layout.Sider属性 说明
className/style/width（200）
breakpoint 触发响应式布局的断点{md：’768px’}
collapsed 当前收起状态
collapseWidth 收缩宽度，为0出现特殊trigger
collapsible 是否可收起
defaultCollapsed 是否默认收起
reverseArrow 折叠提示箭头方向，Sider左边时可用
trigger 自定义trigger，设置null，隐藏trigger
onCollapse 展开收起的回调（collapse, type）=> {}

属性 说明

offsetTop 距窗口顶部。。。。
target 监听滚动事件的元素，返回对应DOM的函数
onChange 固定状态改变时触发的回调
