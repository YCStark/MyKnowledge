#Windows

代码编辑

代码跳转：Ctrl + 左键，Ctrl + B，函数、变量申明

调用位置：Alt + F7，查找调用者

自动补全：Setting》Editors》general》Code Completing：Autopopup

打开类：Ctrl + N

超级打开：双击shift

复制整行：Ctrl + D

删除整行：Ctrl + Y

折叠/展开：Ctrl + -/+

选择：Ctrl + W， 第一次选中word，第二次表达式，第三次函数

重构

改名：Shift + F6， 同时修改引用的位置

移动文件：F6， 修改文件引用位置

抽取函数：Ctrl+ Alt + M，整块代码抽取成函数

抽取变量：Ctrl + Alt + V， 当前选中抽取为变量

移动整块代码：Ctrl + Shift + 上下

包裹：Ctrl + Alt + T， 外层包裹，如：if、try等

NPM

View》Tool Windows》 npm

快捷键

Ctrl + Shift + A ，通过关键字检索到对应功能及其快捷键

格式化代码

Ctrl + Shift + L， 可设置为commit前自动格式化
