#一、 按钮

##1.1 Button

按钮间隙：React不输出元素键换行符，同一行按钮可能紧挨着，为保持间隙，可以放在ButtonToolbar中。

按钮标签：DOM根据你提供的属性自动选择，有href，按钮使用a元素，否则button

跨浏览器兼容：当连接标签渲染为a时，不能被disabled，需要pointer-events:none; 让链接阻止被聚焦，最新浏览器支持

Button属性 说明
Active 默认false
Block 作为块状元素
bsClass 默认btn
bsSize lg/large/sm/small/xs/xsmall
bsStyle 默认default，success/warning/danger/info/primary/link
componentClass/disabled/href/onClick
type 默认button, reset/submit。 HTML按钮类型属性

##5.1 FormGroup、FormControl、COntrolLabel

FormControl将一个表单控件渲染成bootstrap样式，FormGroup以相应留白封装Form控件，支持标签、帮助内容、校验状态，为可存取，需设置FormGroup中的controlId，使用ControlLabel设定标签

​ FormControl直接渲染input或其他组件，如需存取未受控的FormControl值，需设定ref属性，调用ReactDOM.findDOMNode（ref）取出对应节点。

​ 组件包含大量FormGroup，最好构造新组件封装。

FormGroup属性 说明

bsClass 默认form-group
controlId 设置FormControl的id，以及FormControl.Label的htmlFor

id

type 类名为input，使用
